# Brilliantwear

Visit at: https://www.brilliantwear.se/

## Full Stack Web Development Technologies used:  
* HTML
* CSS
* JavaScript
* Node.js
* Express
* MongoDB
* JSON Web Token
* Docker & Swarm mode
* Nginx
* React.js
* TypeScript
* Bootstrap 5
* Next.js 13 (App router)
* Tailwind CSS
